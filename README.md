## How to use

1. Pick an algorithm or problem.
2. Implement it in a descriptively named file.

## File & commit conventions

- File: use lowercase + underscores, e.g. `merge_sort.cpp`
- Function names: `camelCase` or `snake_case` consistently
- Tests/examples: keep simple `main()` examples at the bottom of each file
